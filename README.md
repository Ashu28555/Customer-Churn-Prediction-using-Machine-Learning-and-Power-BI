# Customer Churn Prediction using Machine Learning and Power BI

An end-to-end Machine Learning project that predicts customer churn and provides interactive Power BI dashboards for business analysis and model performance monitoring.

---

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Gradient Boosting
- Matplotlib
- Seaborn
- Power BI

---

## Project Workflow

Customer Dataset
      ↓
Data Cleaning
      ↓
EDA
      ↓
Feature Engineering
      ↓
Model Training
      ↓
Model Evaluation
      ↓
Customer Churn Prediction
      ↓
Power BI Dashboard

---

## Machine Learning Model

Best Model:
- Gradient Boosting

Model Accuracy:
- 91.74%
